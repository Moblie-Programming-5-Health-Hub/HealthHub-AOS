## 커밋 

- `feat` : 새로운 기능 구현
- `mod` : 코드 수정
- `add` : feat 이외의 부수적인 코드 추가, 라이브러리 추가
- `del` : 쓸모없는 코드 삭제
- `chore` : 변수명 및 함수명 수정과 같은 사소한 수정
- `fix` : 버그 수정
- `docs` : 문서 추가, 수정, 삭제
- `init` : 프로젝트 초기 세팅
